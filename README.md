# Job Board Assignment

## Tech Stack

Frontend:
- React
- TypeScript
- Axios

Backend:
- NestJS
- TypeScript

## Features

- View Jobs
- Add Jobs
- Search Jobs
- Form Validation
- REST API (GET /jobs, POST /jobs)

## Run Frontend

cd frontend / 
npm install / 
npm run dev

## Run Backend

cd backend / 
npm install / 
npm run start:dev
